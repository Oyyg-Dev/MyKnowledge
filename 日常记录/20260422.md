● 获取 GitHub Personal Access Token                                                                                                                            
                                                                                                                                                               
  步骤：                                                                                                                                                       
                                                                                                                                                               
  1. 打开浏览器，登录 GitHub，进入：                                                                                                                           
  Settings → Developer settings → Personal access tokens → Fine-grained tokens                                                                                 
                                                                                                                                                               
  1. 直达链接：https://github.com/settings/tokens?type=beta                                                                                                    
  2. 点击 Generate new token                                                                                                                                   
  3. 填写基本信息：                                                                                                                                            
    - Token name：随便起个名，比如 claude-code-mcp                                                                                                             
    - Expiration：选择有效期（建议 90 天，到期可以重新生成）                                                                                                   
    - Repository access：选 All repositories 或指定你需要的仓库                                                                                                
  4. 在 Permissions 中，根据需要勾选权限，常用的：                                                                                                             
                                                                                                                                                               
  | 权限          | 级别         | 用途                 |                                                                                                      
  |---------------|--------------|----------------------|                                                                                                      
  | Contents      | Read & Write | 读写仓库文件         |                                                                                                      
  | Issues        | Read & Write | 管理 issue           |                                                                                                      
  | Pull requests | Read & Write | 管理 PR              |                                                                                                      
  | Metadata      | Read-only    | 基础仓库信息（必选） |                                                                                                      
                                                                                                                                                               
  5. 点击 Generate token，复制生成的 token（以 github_pat_ 开头）                                                                                              
                                                                                                                                                               
  ★ Insight ─────────────────────────────────                                                                                                                  
  - Fine-grained token 比 Classic token 更安全，可以精确控制到单个仓库和具体权限                                                                               
  - Token 只在生成时显示一次，关掉页面就看不到了，务必立刻复制保存                                                                                             
  - 如果你只是读取公开仓库，权限可以给最小的 read-only 就够了                                                                                                  
  ─────────────────────────────────────────────                                                                                                                
                                                                                                                                                               
  拿到 token 后告诉我，我帮你配到 settings 里。你想装全局还是当前项目？
