# 生产计件系统 - 项目概述

## 📋 项目信息

- **项目编号**: B001
- **项目名称**: 生产计件系统
- **项目状态**: 🟡 持续开发中
- **技术栈**: uni-app 小程序 + 若依框架 + Java 8 + MySQL

## 🎯 项目目标

为生产车间提供便捷的计件管理系统，支持：
- 工人通过小程序提交计件数据
- 班组长审核计件单据
- 管理层查看产量统计和报表
- 与 OA 系统、T100 ERP 系统集成

## 🔑 核心功能模块

### 1. 计件提交（小程序）
- 扫描工单号
- 选择工种和工序
- 选择参与人员
- 填写产量数据
- 提交计件单据

### 2. 计件审核（小程序 + PC）
- 班组长审核计件单据
- 支持驳回和修改
- 审核记录追溯

### 3. 产量查询（小程序 + PC）
- 按日期、工单、人员查询
- 按大类、工序筛选
- 按机台-组别筛选
- 数据导出

### 4. 工资计算（PC）
- 根据计件数据计算工资
- 支持不同工种的计算规则
- 生成工资报表

### 5. 数据统计（PC）
- 产量统计
- 人员效率分析
- 班组绩效对比

## 📊 业务场景

### 场景 1: 普通计件
1. 工人扫描工单号
2. 系统自动带出料件和工序信息
3. 工人选择参与人员
4. 填写产量数据
5. 提交给班组长审核

### 场景 2: 无工单计件
1. 工人选择"无工单"
2. 可扫工单号仅展示信息（方便审核）
3. 手动选择工序
4. 填写产量数据
5. 提交审核

### 场景 3: 支援人员计件
1. 支援人员可选择其他班组的工序
2. 通过"支援工序"功能切换
3. 工资计算以人的部门班组为准

### 场景 4: 总装班特殊处理
1. 区分压装和包装两个班组
2. 选择人员时需要区分班组
3. 支持选择上次填写的机台-组别信息（前15天）
