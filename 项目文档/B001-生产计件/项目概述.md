### 使用工具
项目提供 `scripts/connection_helper.py` 工具，自动从全局配置读取连接信息：

```bash
# 查看数据库配置
python scripts/connection_helper.py db ms

# 查看 SSH 配置
python scripts/connection_helper.py ssh t100
```

### Python 脚本中使用
```python
from scripts.connection_helper import get_db_config, get_ssh_config

# 获取数据库配置
db_config = get_db_config("ms")
print(f"Host: {db_config['host']}, User: {db_config['username']}")

# 获取 SSH 配置
ssh_config = get_ssh_config("t100")
print(f"SSH: {ssh_config['ssh_alias']}")
```

## ⚠️ 关键规则

### 1. API 修改同步检查（重要！）
修改后端 API 时，**必须**检查以下位置是否需要同步：
- PC 端：`faretms-ui/src/api/`
- 小程序：`minicard/MiniCardVue/common/api/`

**检查内容**：
- ✅ 接口路径是否一致
- ✅ 参数结构是否匹配
- ✅ 返回格式是否统一

### 2. 创新平台模块特殊规则
- ✅ 提案用 `userNo/userName`
- ✅ 项目用 `applicant/applicantName`
- ❌ 禁止用 `createBy/createName` 获取真实用户信息
- **原因**: 创新平台同步 OA 数据时，create 字段为 admin 管理账户名，非真实用户

### 3. OA 回调接口
- ✅ 必须做空值防御
- ✅ 必须保证幂等性

### 4. 后端开发
- ✅ 代码修改后必须编译通过：`mvn clean package -DskipTests`

### 5. 功能限制
- ❌ 不得新增导出功能（export.js / 导出接口），除非明确批准
- **说明**: 此规则特指创新平台模块中的小程序页面操作，不涉及生产计件等其他模块
