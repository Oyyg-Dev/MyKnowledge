## 📝 开发规范

### 前端规范
- 遵循 Vue 2 风格指南
- 使用 Element UI 组件库
- 统一使用 Vue CLI 构建工具

### 后端规范
- RESTful API 设计
- 统一返回格式
- 异常统一处理

### 数据库规范
- **重要**: 查询 T100 数据库时必须使用 `DSDATA.表名`
- 示例: `SELECT * FROM DSDATA.bas_commodity`
- 或者先执行: `ALTER SESSION SET CURRENT_SCHEMA = DSDATA;`
