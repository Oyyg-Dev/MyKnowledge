## 💾 数据库连接

### 连接信息
读取全局资源目录 `C:\Users\Administrator\.claude\connections.json` 获取连接详情。

| 资源 | 引用键 | 说明 |
|------|--------|------|
| 数据库 T100 测试区 | `databases.t100_toptst` | 只读账号 T100_RO ✅ |
| 数据库 T100 正式区 | `databases.t100_topprd` | 只读账号 T100_RO_PROD ✅ |
| SSH falada | `ssh.falada` | falada_ssh_ro@192.168.1.177 |

⭐ **重要**: T100 业务数据存储在 `DSDATA` 模式，查询时必须使用 `DSDATA.表名`

### 使用工具
项目提供 `.claude/connection_helper.py` 工具，自动从全局配置读取连接信息：

```bash
# 查看数据库配置
python .claude/connection_helper.py db t100_topprd

# 查看 SSH 配置
python .claude/connection_helper.py ssh falada
```

### Python 脚本中使用
```python
import sys
sys.path.append('.claude')
from connection_helper import get_db_config, get_ssh_config, build_ssh_command

# 获取数据库配置
db_config = get_db_config("t100_topprd")
print(f"Host: {db_config['host']}, User: {db_config['username']}")

# 构建 SSH 命令
ssh_cmd = build_ssh_command("falada", "ls /home/rearEnd/act-admin-starfall/logs")
print(ssh_cmd)  # ssh falada_ssh_ro "ls /home/rearEnd/act-admin-starfall/logs"
```

### 数据库查询示例
```sql
-- T100 数据库查询（必须使用 DSDATA 模式）
SELECT * FROM DSDATA.pmaa_t WHERE ROWNUM <= 10;

-- 或者先切换模式
ALTER SESSION SET CURRENT_SCHEMA = DSDATA;
SELECT * FROM pmaa_t WHERE ROWNUM <= 10;
```
