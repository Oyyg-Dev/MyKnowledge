## 💾 数据库连接

### 连接信息
读取全局资源目录 `C:\Users\Administrator\.claude\connections.json` 获取连接详情。

| 资源 | 引用键 | 说明 |
|------|--------|------|
| 数据库 wms | `databases.wms` | MariaDB，只读账号 wms_ro ✅ |
| SSH wms | `ssh.wms` | wms_ssh_ro@192.168.1.187 |

### 使用工具
项目提供 `.claude/connection_helper.py` 工具，自动从全局配置读取连接信息：

```bash
# 查看数据库配置
python .claude/connection_helper.py db wms

# 查看 SSH 配置
python .claude/connection_helper.py ssh wms
```

### Python 脚本中使用
```python
import sys
sys.path.append('.claude')
from connection_helper import get_db_config, get_ssh_config, build_ssh_command

# 获取数据库配置
db_config = get_db_config("wms")
print(f"Host: {db_config['host']}, User: {db_config['username']}")

# 构建 SSH 命令
ssh_cmd = build_ssh_command("wms", "ls /opt/ilc-wms-prod/app_backend/platform/log")
print(ssh_cmd)
```

### 数据库查询示例
```bash
# 通过 SSH 使用 MariaDB 客户端查询
ssh wms_ssh_ro "/opt/mariadb/bin/mysql -u wms_ro -p'Faret-123666' -e 'SHOW DATABASES'"

# 查询特定数据库
ssh wms_ssh_ro "/opt/mariadb/bin/mysql -u wms_ro -p'Faret-123666' ilc_wms_prod -e 'SELECT COUNT(*) FROM user'"
```
