## 一、数据库表设计（XX张表）
###  提案表 `t_proposal`
| 字段名 | 类型 | 说明 | 索引 |
| --- | --- | --- | --- |
| id | BIGINT | 主键ID | PK |
| title | VARCHAR(200) | 提案标题 | IDX |
| dept | VARCHAR(50) | 所属部门 | IDX |
| main_category_code | VARCHAR(50) | 大类编码（单选） | IDX |
| sub_category_codes | TEXT | 小类编码(JSON数组,多选) | - |
| tags | VARCHAR(255) | 技术标签(逗号分隔,兼容旧数据) | - |
| problem | TEXT | 问题描述 | - |
| innovation | TEXT | 创新方案 | - |
| feasibility | TEXT | 技术可行性 | - |
| cost_saving | VARCHAR(50) | 成本节省 | - |
| efficiency_gain | VARCHAR(50) | 效率提升 | - |
| quality_improvement | VARCHAR(50) | 质量提升 | - |
| other_value | TEXT | 其他价值 | - |
| budget | DECIMAL(10,2) | 预算(万元) | - |
| manpower_count | INT | 人力投入人数 | - |
| manpower_duration | INT | 人力投入月数 | - |
| start_date | DATE | 开始日期 | - |
| end_date | DATE | 结束日期 | - |
| status | VARCHAR(20) | 状态 | IDX |
| created_by | BIGINT | 创建人ID | IDX |
| oa_process_id | VARCHAR(50) | OA流程ID | UK |
| submitted_at | DATETIME | 提交时间 | IDX |
| created_at | DATETIME | 创建时间 | - |
| updated_at | DATETIME | 更新时间 | - |


**状态枚举：**

+ `Draft`: 草稿
+ `Submitted`: 已提交
+ `OAProcessing`: OA审批中
+ `OAApproved`: OA已通过
+ `OARejected`: OA已驳回

**类别字段说明：**

1. **main_category_code**（大类编码，单选）
    - 存储格式：字符串
    - 示例：`"TECH_INNOVATION"`
    - 外键关联：`t_category.code`
2. **sub_category_codes**（小类编码，多选）
    - 存储格式：JSON数组
    - 示例：`["TECH_AUTOMATION", "TECH_VISION"]`
    - 外键关联：`t_category.code`

**索引设计：**

```sql
CREATE INDEX idx_status_dept ON t_proposal(status, dept);
CREATE INDEX idx_main_category ON t_proposal(main_category_code);
CREATE INDEX idx_created_by ON t_proposal(created_by);
CREATE INDEX idx_submitted_at ON t_proposal(submitted_at DESC);
```

**示例数据：**

```json
// 完整示例
{
  "main_category_code": "TECH_INNOVATION",
  "sub_category_codes": ["TECH_AUTOMATION", "TECH_VISION"]
}
```

**业务规则：**

1. 大类编码必选，且只能选择一个
2. 小类编码至少选择一个，可多选
3. 切换大类时，已选的小类自动清空
4. 后端需验证：
    - 大类编码是否存在且level=1
    - 小类编码是否存在且parent_code=大类编码

**查询示例：**

```sql
-- 按大类查询
SELECT p.*, c.name as main_category_name 
FROM t_proposal p
LEFT JOIN t_category c ON p.main_category_code = c.code
WHERE p.main_category_code = 'TECH_INNOVATION';

-- 按小类查询（MySQL 5.7+）
SELECT * FROM t_proposal 
WHERE JSON_CONTAINS(sub_category_codes, '"TECH_AUTOMATION"');
```
